# react-native-nuemorphism

Neumorphism UI Component Library for React Native

## Installation

```sh
npm install react-native-nuemorphism
```

## Usage

```js
import { multiply } from "react-native-nuemorphism";

// ...

const result = await multiply(3, 7);
```
